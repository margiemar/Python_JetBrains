Translator.py translates any word from and to languages:

"Arabic", "German", "English", "Spanish", "French", "Hebrew", "Japanese",
"Dutch", "Polish", "Portuguese", "Romanian", "Russian", "Turkish".


Script gets an argumets like "english spanish hello" - from, to languages and word for translate.

If destination language is "all", script translaite to all that languages.

Result will be saved in file with input word as name.txt.
